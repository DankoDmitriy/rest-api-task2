#Task 2: REST API - Base.

#### Instruction to run

- Download project 
- Run maven clean install
- Run maven deploy
- Run Spring MVC application in module WEB
- Wait for successful log
- Do requests a localhost:8080

You need for run this application:
- IntelliJ IDEA
- Java 1.8
- Spring 5.3.12
- Mysql 8.0+
